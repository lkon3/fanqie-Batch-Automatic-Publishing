# 快速修复指南 - ChromeDriver 问题

## 问题原因
webdriver-manager 自动下载的 ChromeDriver 版本不匹配或损坏。

## 解决方案（3分钟完成）

### 步骤 1: 确认 Chrome 版本
1. 打开 Chrome 浏览器
2. 地址栏输入: `chrome://version/`
3. 记下版本号，例如: `130.0.6723.117`

### 步骤 2: 下载 ChromeDriver

**方法 A - 使用新版本下载地址（推荐）**

1. 访问: https://googlechromelabs.github.io/chrome-for-testing/
2. 找到 "Stable" 的最新版本号
3. 点击下载链接，选择: `win64` 版本
4. 下载文件名类似: `chromedriver-win64.zip`

**方法 B - 直接使用最新稳定版**

1. 直接访问下载地址:
   https://storage.googleapis.com/chrome-for-testing-public/LATEST_RELEASE_WIN64/download

2. 或者访问这个页面选择版本:
   https://chromedriver.chromium.org/downloads

### 步骤 3: 解压并安装

1. 找到下载的 `.zip` 文件
2. 解压（右键 → 解压到...）
3. 在解压后的文件夹中找到 `chromedriver.exe`
4. 将 `chromedriver.exe` 复制到项目目录:
   ```
   C:\Users\1\Desktop\番茄自动发布\chromedriver.exe
   ```

### 步骤 4: 验证安装

打开命令行，运行:
```bash
cd C:\Users\1\Desktop\番茄自动发布
python test_fix.py
```

如果看到浏览器打开并显示百度页面，说明安装成功！

### 步骤 5: 使用程序

```bash
python main.py
```

选择 `3. 测试登录` 即可。

## 常见问题

### Q: 下载后还是提示 "不是有效的 Win32 应用程序"？
A:
1. 确认下载的是 **win64** 版本（不是 win32）
2. 确认您的系统是 64 位（大多数现代电脑都是）
3. 如果还是不行，尝试重启电脑

### Q: 不知道下载哪个版本？
A:
- Chrome 版本 115+: 使用方法 A 的下载地址
- Chrome 版本 114 以下: 使用方法 B

### Q: 下载的文件打不开？
A:
- 不要直接打开 zip 文件
- 先解压，然后复制里面的 chromedriver.exe
